# ImaginAI - テーブル定義

## history (履歴エントリ)

保存方式: ファイルベース (履歴保存ディレクトリ配下の `images/` に 1 エントリ = 1 メディアファイル + 1 JSON メタデータ)。
スキーマは provider 別サブオブジェクト構造に統一されている。旧スキーマ (フラットフィールド) は起動時のロード時に自動マイグレーションされ、書き戻される。

### 共通フィールド (両プロバイダ)

| カラム名 | 型 | 説明 |
|---|---|---|
| id | string (UUID) | 主キー — 一意識別子 |
| createdAt | string (ISO 8601) | エントリ作成日時 |
| updatedAt | string (ISO 8601) | 最終更新日時 |
| provider | "gemini" \| "openai" | 生成プロバイダ。旧データは model id から推定して埋められる |
| model | string | 生成に使用した API モデル ID |
| modelDisplayName | string | UI 表示用のモデル名 |
| mediaType | "image" \| "video" \| "music" \| "voice" | メディア種別 |
| prompt | string | 生成プロンプト |
| numberOfImages | number | 生成枚数 (1-10) |
| referenceImagePaths | string[] | 使用した参照画像のフルパス配列 |
| generatedImagePaths | string[] | 生成ファイルのフルパス配列 (画像: .png/.jpg/.webp、動画: .mp4、音楽/音声: .mp3 もしくは .wav) |
| imageWidth | number? | 生成画像の幅 (ピクセル、画像のみ) |
| imageHeight | number? | 生成画像の高さ (ピクセル、画像のみ) |
| fileSize | number? | 生成ファイルのサイズ (バイト) |
| elapsedMs | number? | 生成所要時間 (ミリ秒、API 呼び出し開始から完了まで) |
| editMode | boolean | 画像編集モードで生成されたか (true なら添付の 1 枚目を編集対象として処理した) |

### gemini サブオブジェクト (provider === "gemini" の時のみ存在)

| カラム名 | 型 | 説明 |
|---|---|---|
| gemini.negativePrompt | string | ネガティブプロンプト |
| gemini.aspectRatio | GeminiAspectRatio | アスペクト比 ("1:1", "16:9" 等) |
| gemini.quality | GeminiQuality | 出力解像度 ("512px", "1k", "2k", "4k") |
| gemini.videoDuration | number? | 動画の長さ (秒、動画のみ: 4, 6, 8) |
| gemini.videoResolution | string? | 動画の解像度 (動画のみ: "720p", "1080p", "4k") |
| gemini.seed | number? | シード値 (動画のみ、再現性のため) |
| gemini.audioTexts | string[]? | 音楽/音声生成時に API が返したテキストパート配列 (音楽は歌詞/楽曲説明、音声は補足テキスト) |
| gemini.styleInstruction | string? | TTS 生成時に使用したスタイル指示文 (英語) |
| gemini.voice | string? | TTS 生成時に使用したボイス名 (例: "Kore") |

### openai サブオブジェクト (provider === "openai" の時のみ存在)

| カラム名 | 型 | 説明 |
|---|---|---|
| openai.size | OpenAIImageSize | "1024x1024" / "1024x1536" / "1536x1024" / "2048x2048" / "2048x1152" / "3840x2160" / "2160x3840" |
| openai.quality | "low" \| "medium" \| "high" | 出力品質 |
| openai.outputFormat | "png" \| "jpeg" \| "webp" | 出力フォーマット |
| openai.background | "opaque" \| "transparent" | 背景 (gpt-image-2 では "opaque" 固定) |
| openai.inputFidelity | "low" \| "high" | 編集時の入力忠実度 (gpt-image-1 では未使用) |

### ストレージ構造

```text
<historyDir>/
  images/
    <uuid>.png             # 生成画像 (PNG)
    <uuid>.jpg             # 生成画像 (JPEG、OpenAI の output_format=jpeg)
    <uuid>.webp            # 生成画像 (WebP、OpenAI の output_format=webp)
    <uuid>.mp4             # 生成動画 (Veo)
    <uuid>.mp3             # 生成音楽/音声 (通常)
    <uuid>.wav             # 生成音声 (TTS、ffmpeg フォールバック)
    <uuid>.json            # HistoryEntry メタデータ
  thumbnails/
    <uuid>.jpg             # サムネイル (JPEG、長辺最大 300px)
```

- 1 メディアファイル = 1 UUID。複数枚生成時は UUID も複数。
- 拡張子は API レスポンスの mimeType (OpenAI) または後処理結果 (Gemini TTS の PCM → MP3/WAV) に応じて動的に決まる。
- 動画サムネイルは ffmpeg により 1 秒目のフレームから抽出。ffmpeg 不在時はサムネイルなし。
- 音楽/音声エントリにはサムネイル画像なし (UI ではアイコンで表示)。

## settings (アプリケーション設定)

保存方式: JSON ファイル (`<userData>/settings.json`)

| カラム名 | 型 | 説明 |
|---|---|---|
| language | string | UI 言語 ("ja" または "en") |
| theme | string | テーマ設定 ("light", "dark", "system") |
| historyDir | string | 履歴保存ディレクトリのフルパス |

## api-keys (API キー、スキーマ v2)

保存方式: 暗号化ファイル (`<userData>/api-keys.enc`、Electron safeStorage 使用)。
旧スキーマ (v1 以前、フラットな defaultKey / freeTierKey / customs) は起動時のロード時に自動的に v2 へ変換され、変換前のバイナリは `api-keys.enc.bak.v1` としてバックアップされる。

### トップレベル

| カラム名 | 型 | 説明 |
|---|---|---|
| schemaVersion | 2 | スキーマバージョン (常に 2) |
| providers | Record<"gemini" \| "openai", ProviderKeySet> | プロバイダ別のキー集合 |
| activeId | string | アクティブなキー識別子。形式: `"<provider>:<slot>"` (例: `"gemini:default"`, `"openai:custom:0"`) |

### ProviderKeySet (各プロバイダ)

| カラム名 | 型 | 説明 |
|---|---|---|
| default | ApiKeySlot | 既定キー |
| freeTier | ApiKeySlot | フリーティアキー (Gemini のみ UI 表示。OpenAI では空スロットを構造保持のため維持) |
| customs | ApiKeySlot[] | カスタムキー (最大 5 件、ユーザーが任意のタイトルを付与) |

### ApiKeySlot (キー枠)

- `key` (string): API キー文字列 (未設定なら空文字)
- `isFreeTier` (boolean): フリーティア扱いかどうか。Gemini のみ意味があり、OpenAI では常に false
- `title` (string?): カスタムキーの表示名 (default/freeTier では未使用)
